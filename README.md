# OCR Фото Аналізатор

Цей сайт дозволяє користувачам завантажувати фото та аналізувати текст на них за допомогою нейронної мережі (Tesseract.js).

## Функціонал
- Перетягування фото на сайт
- Аналіз тексту на фото
- Завантаження результату як текстового файлу

## Як запустити локально
Відкрийте `index.html` у браузері.

## Деплой на Vercel
1. Створіть репозиторій на GitHub.
2. Запуште цей проект до репозиторію.
3. Підключіть репозиторій до Vercel для автоматичного деплою.

## Технології
- HTML, CSS, JavaScript
- Tesseract.js для OCR
