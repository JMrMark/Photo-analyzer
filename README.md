# OCR Фото Аналізатор

Цей сайт дозволяє користувачам завантажувати фото та аналізувати текст на них за допомогою нейронної мережі (Tesseract.js).

## Функціонал
- Перетягування фото на сайт
- Аналіз тексту на фото
- Завантаження результату як текстового файлу
- Вибір моделі OCR
- Темний/світлий режим
- Система ліміту для звичайних користувачів (5 фото за 5 хв)
- VIP-режим для користувачів, які увійшли через Google

## Як запустити локально

Оскільки браузери не дозволяють завантажувати модулі з `file://` протоколу, потрібно запустити локальний веб-сервер.

### Варіант 1: Python (якщо встановлено)
```bash
# Відкрийте термінал/PowerShell в цій папці
python -m http.server 8000
```
Потім відкрийте http://localhost:8000 у браузері.

### Варіант 2: Використайте bat/ps1 скрипт
**Windows CMD:**
```bash
run-server.bat
```

**Windows PowerShell:**
```bash
.\run-server.ps1
```

### Варіант 3: Node.js
```bash
npx http-server
```

## Деплой на Vercel

1. Створіть репозиторій на GitHub.
2. Запуште цей проект до репозиторію.
3. Підключіть репозиторій до Vercel для автоматичного деплою.
4. Vercel автоматично розпізнає статичний сайт та розгорне його.

## Технології
- HTML, CSS, JavaScript
- Tesseract.js для OCR
- Google Identity Services для входу
- LocalStorage для збереження налаштувань та квот

## Структура проекту
- `index.html` - основна сторінка
- `styles.css` - стилі
- `app.js` - основний JavaScript (для локального тестування)
- `js/` - модульна версія для Vercel (опціонально)
